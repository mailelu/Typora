# 创建版本库

创建一个新的文件夹

使用`git init`来设置为仓库

**注意不要使用中文**

此时目录下会多出一个git文件夹,作为重要的git系统文件

```git
git add 	//将文件加入到暂存区
git commit -m <message>   //将文件加入到工作区
```
